# VMSC E-Name Cards

Danh thiếp điện tử / E-Name Cards — Tổng công ty Bảo đảm an toàn hàng hải Việt Nam.

## Các trang
- `index.html` — Đỗ Văn Lâm (trang chính / root URL)
- `lam.html`   — Đỗ Văn Lâm
- `chinh.html` — Nguyễn Phúc Chính

## Triển khai bằng GitHub Pages
1. Tạo repository mới (ví dụ: `namecard`), để Public.
2. Upload toàn bộ file trong thư mục này (Add file -> Upload files -> kéo cả 3 file -> Commit).
3. Vào **Settings -> Pages**.
4. Mục **Source**: chọn `Deploy from a branch`, branch `main`, thư mục `/ (root)` -> **Save**.
5. Đợi ~1 phút. Link sẽ là: `https://<tên-tài-khoản>.github.io/namecard/`
   - Thẻ của Lâm:   `https://<tên-tài-khoản>.github.io/namecard/`
   - Thẻ của Chính: `https://<tên-tài-khoản>.github.io/namecard/chinh.html`

## Triển khai bằng Netlify Drop (nhanh, tạm thời)
1. Mở https://app.netlify.com/drop
2. Kéo–thả CẢ THƯ MỤC này vào vùng thả.
3. Nhận link ngay. Thẻ của Lâm = link gốc; thẻ của Chính = `/chinh.html`.
